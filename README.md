# CircaLife - Advanced Body Optimization App

![CircaLife Banner](https://via.placeholder.com/800x200/0A0E27/4DDBFF?text=CircaLife+Body+Optimization)

## 🌟 Overview

**CircaLife** adalah aplikasi mobile revolusioner yang menggabungkan riset ilmiah terkini tentang optimasi tubuh manusia. Aplikasi ini dibangun berdasarkan:

- 🏆 **Nobel Prize Research** (2017) - Chronobiology & Circadian Rhythm
- 🔬 **Latest Autophagy Science** - Cellular renewal & longevity
- 💓 **Vagus Nerve Stimulation** - HRV & stress recovery
- 📚 **Evidence-based protocols** - Dari dokumen panduan optimasi tubuh

## 🔬 Scientific Foundation

### Nobel Prize Chronobiology (2017)
```
Jeffrey Hall, Michael Rosbash, Michael Young
→ Mekanisme molekuler circadian rhythm
→ PER protein oscillation (24h cycle)
→ Mengontrol: hormon, tidur, suhu, metabolisme
```

### Autophagy & Longevity Research
```
Cell Death Discovery (2025) - Mitochondrial Autophagy
Nature Aging (2021) - Autophagy in healthy aging
Frontiers Endocrinology (2022) - Autophagy & Longevity

Key findings:
→ Autophagy menurun dengan usia
→ Fasting/exercise mengaktifkan AMPK
→ Mitophagy = kunci anti-aging
→ NAD+ decline 50% di usia 40
```

### Vagus Nerve & HRV
```
PMC Studies (2020-2025) - Vagus Nerve Stimulation
→ 80% vagus signals: gut → brain (afferent)
→ HRV = proxy of vagal tone
→ taVNS (transcutaneous auricular) efektif
→ Cold exposure, breathing, yoga meningkatkan vagal tone
```

## 🚀 Installation

### Prerequisites
- Node.js 18+ 
- Expo CLI
- Android Studio / Xcode (untuk testing device)

### Setup

```bash
# Clone repository
git clone https://github.com/yourusername/circalife.git
cd circalife

# Install dependencies
npm install

# Start Expo development server
npm start

# Run on Android
npm run android

# Run on iOS
npm run ios

# Run on Web
npm run web
```
